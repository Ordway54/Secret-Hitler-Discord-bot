# Secret-Hitler-Discord-bot
## Overview
A Discord bot application written in Python 3 and using the discord.py library. This bot application allows the user to play games of *Secret Hitler* entirely within the Discord client.

## About the Game
*Secret Hitler* is a social role deduction board game created by Mike Boxleiter, Tommy Maranges, and Mac Schubert.
